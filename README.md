# Online Examination & Proctoring System

Simple modular Java Swing project for a college-level online examination and proctoring system.

Current setup:
- GUI: Swing
- Storage: file handling
- Proctoring log: file-based (`data/activity_log.txt`)

Data files:
- `data/users.json`
- `data/questions.txt`
- `data/results.txt`
- `data/activity_log.txt`

Default login data:
- Admin: `admin` / `admin123`
- Student: `student1` / `stud123`
